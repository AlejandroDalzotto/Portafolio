# Varano

**Varano** es una aplicación diseñada para la gestión de datos en instituciones educativas. Este proyecto busca simplificar el manejo de información académica mediante una interfaz intuitiva, permitiendo a los usuarios registrar y consultar datos de manera eficiente y segura.

## Descripción

Varano facilita la carga y gestión de datos académicos, como las notas de examen, las materias y los datos de los estudiantes. Su objetivo es proporcionar una herramienta centralizada y fácil de usar para el manejo de la información educativa. El sistema está construido de manera escalable y en futuras versiones se implementarán características adicionales, como la autenticación basada en roles para la gestión de usuarios.

## Funcionalidades Clave

- **Carga de Datos**: Permite registrar información sobre estudiantes, materias, y notas de examen de manera sencilla.
- **Filtrado y Búsqueda**: Herramientas para buscar y filtrar rápidamente los datos registrados.
- **Gestión de Usuarios (Futuro)**: En versiones futuras, se implementará un sistema de autenticación basado en roles para diferentes niveles de acceso.

## Tecnologías Utilizadas

- **Java**: Lenguaje de programación principal utilizado para el desarrollo del Back-end.
- **Spring Framework**: Framework que facilita la creación de aplicaciones empresariales robustas y seguras.
- **MySQL**: Base de datos relacional utilizada para almacenar y gestionar los datos de la aplicación.

## Objetivo

El propósito de **Varano** es facilitar la gestión académica en instituciones educativas, mejorando la eficiencia en la carga, búsqueda y filtrado de información. Este proyecto está en una fase preliminar, con funcionalidades básicas de registro de datos. A medida que se identifiquen nuevos requerimientos, se planea añadir nuevas características, como la gestión avanzada de usuarios.
